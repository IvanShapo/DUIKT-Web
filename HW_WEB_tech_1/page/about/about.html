Про мене

Я Шаповалов Іван Олексійович, junior Front-end розробник та системний адміністратор з практичним досвідом роботи у веб-розробці та адмініструванні систем Windows і Linux. Маю досвід створення адаптивних інтерфейсів на React і Vue, використовуючи TypeScript, SCSS, Tailwind та MUI, а також інтеграцію з REST API через Axios і управління станом через Redux та Vuex.
